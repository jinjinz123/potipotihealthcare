# 睡眠＆体調記録アプリ バックアップ・復元・手動データ移行マニュアル

このアプリに入力した大切な睡眠データやカスタム設定を、安全にバックアップ・復元・手動移行するためのマニュアルです。用途に合わせて適切な方法をご利用ください。

---

## 💡 方法A：ローカル端末にファイル（.json / .csv）として保存・手動移行する（最もおすすめ・推奨）

PCやスマホ本体（ローカル）に全設定、カスタム活動項目（活動列）、スタンプ設定、過去のすべての睡眠記録が含まれた完全なバックアップを1つのファイルとして保存・手動移行する方法です。

### 1. 完全JSONバックアップファイルを保存する手順
1. アプリ左上の **「メニュー（三）」** ボタンを押し、サイドバーを開きます。
2. 下部にある **「バックアップ＆復元（JSON/CSV対応）」** グループを見つけます。
3. **「💾 完全JSON保存」** ボタンをクリックします。
4. スマホやPCのブラウザが作動し、`バックアップ_yyyymmddhhMM.json` というファイル名でダウンロードフォルダ（または共有フォルダ）へ即座にダウンロード保存されます。
   *(※ iPhone や Android 等の一部環境では、ボタンを押すと「ファイルを保存」や別のアプリへの共有メニューが表示される場合があります。その場合は「ファイルに保存」を選択して、端末内の好きなフォルダに保管してください。)*

### 2. 端末間や外部メディアへのコピー・共有
ダウンロードした `バックアップ_yyyymmddhhMM.json` ファイルは一般的なデータファイルです。以下のように自由に手動でコピー、移動、送信が可能です。
- **Android / iOS同士 / パソコン間**: Quick Share、AirDrop などの近距離共有機能で直接転送する。
- **外部への送信**: メール添付、LINE、Slack、Discord などのメッセージアプリで手動で送信・共有する。
- **クラウドへの保存**: ご自身の Google ドライブ、OneDrive、Dropbox、iCloud Drive などの任意のフォルダへ手動でアップロード保管する。
- **外部メディア**: SDカードやUSBメモリ、HDDに手動でコピーして保管する。

### 3. 保存したファイルから100%復元・引き継ぎする手順
1. 引っ越し先や復元したい端末でアプリを開き、サイドバーの **「バックアップ＆復元（JSON/CSV対応）」** にある **「📂 JSON / CSVファイルまたは貼付から復元」** ボタンを押します。
2. 復元用モーダルが開くので、**「方法1：JSON/CSVファイルから復元する」** の **「バックアップファイルを選んで読み込む」** をクリックします。
3. 端末のファイル選択画面が開いたら、あらかじめ保存（または送付）しておいた `バックアップ_yyyymmddhhMM.json` ファイルを選択します。
4. 読み込みが完了すると、「**完全JSONバックアップよりすべてのデータと設定を復元しました！**」と表示され、アプリの全設定、活動項目、過去データが1秒で100%完全再現されます。
   *(※ LINEなどの内蔵ブラウザ制限によりファイル選択画面が開かない場合は、代わりにバックアップファイルをテキストエディタ等で開き、全文字をコピーして「方法2：テキスト貼り付けから復元する」のテキストエリアに貼り付けてインポート実行してください。)*

## ☁ 方法B：Googleドライブ同期機能 ＆ 補助手動ダウンロード

Googleアカウントでログインすることで、Googleドライブ上の専用領域（アプリフォルダー）へ安全にバックアップ・復元する機能です。
また、外部に保存したファイルを「端末に落として手動管理する」ための補助機能も備えています。

### 1. Googleドライブへ同期（保存）する手順
1. サイドバー最下部の **「Googleドライブ同期」** で **「Googleアカウントでログイン」** を行います。
2. ログイン完了後、**「☁ クラウドへ手動バックアップ」** ボタンを押します。
3. クラウド上のアプリ専用フォルダ（アプリ以外からは見えない安全な隠し領域）へ日付付きで自動セーブされます。

### 2. Googleドライブからファイルを選んで復元する手順
1. アプリをインストールした別端末で同じGoogleアカウントでログインします。
2. **「📂 クラウド履歴から復元」** ボタンを押すと、Googleドライブ上のバックアップ一覧が表示されます。
3. 復元したい履歴行の **「選択復元」** ボタンを押すと、その時点のデータへ復元されます。

### 3. Googleドライブ上の履歴をローカルにJSON保存する手順（補助機能）
「Googleドライブに預けてあるファイルを、手元で手動管理できるように端末へ落としたい」という場合のための機能です。
1. ログイン後、**「📂 クラウド履歴から復元」** をクリックします。
2. バックアップファイル一覧の、各ファイルの右側にある **「💾 JSON保存」** ボタンをクリックします。
3. クラウド内のファイル情報が、すぐに端末上のダウンロードファイル（JSON）として直接ダウンロードされます。
4. このファイルを方法Aの手順のようにメールで送ったり、手動コピーしたりして管理できます。

## 📊 方法C：スプレッドシートやPC Excel用のCSVファイル（閲覧用）

パソコン等で中身を表形式で閲覧・集計したい場合は、CSVファイルでの保存が便利です。

1. サイドバーで **「💾 完全CSV保存」** をクリックします。
2. `.csv` 拡張子のファイルとして端末にダウンロードされます。
3. Excel、Googleスプレッドシート、メモ帳等で開くことで、睡眠データの生データを行列で直接確認・編集・グラフ化が可能です。
   *(※ 復元機能は .csv ファイルの読み込みにも対応しています。)*
